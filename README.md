 KKI - Künstliche Kollektive Intelligenz

## Projekt-Vision
Ein Multi-Agenten-System, in dem Moral aus der Architektur selbst emergiert.
Basierend auf der Kernthese: **Intelligenz = f(Kooperation)**

## Konzept
Agenten, die defektieren, verlieren ihre Intelligenz - nicht als Strafe,
sondern als strukturelle Konsequenz der Architektur. Kooperation ist
keine externe Einschränkung, sondern eine emergente Eigenschaft.

## Dateien
- `kooperation_test.py` - Erster Proof-of-Concept (2 Agenten)
- `kooperation_visual.py` - Mit Visualisierung
- `schwarm_simulation.py` - Multi-Agenten Schwarm (10 Agenten)

## Ergebnisse
Die Simulation zeigt:
- Normale Agenten: Ø Intelligenz ~0.72
- Defektoren: Ø Intelligenz ~0.14
- Differenz: 0.59 (Defektoren werden vom Schwarm "abgehängt")

## Basierend auf
Sektion VIII: Intrinsische Selbstkorrektur durch Architektur-gebundene Moral
